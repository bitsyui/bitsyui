# @bitsyui/ui-solid

Solid-based interactive components and patterns for Bitsy UI.

Included components:

- `Button`
- `Input`
- `Card`
- `Tabs`
- `Modal`
- `DropdownMenu`

Notes:

- Styles live in `@bitsyui/styles`
- Behavior and interaction logic live here in Solid
- Components use semantic HTML and accessible roles by default
