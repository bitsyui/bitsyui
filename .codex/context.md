# Bitsy UI context

Bitsy UI is a token-led visual system for building across web, apps, and brands.

## Goals

- Themeable across multiple projects
- Portable token system
- CSS-first architecture
- Solid as the first interactive runtime
- Visual docs in Astro
- Sustainable for one maintainer

## Taxonomy

- Foundation
- Primitive
- Composite
- Interactive
- Pattern
- Template

## Non-goals

- Not a huge enterprise framework
- Not utility-class-first
- Not React-first
- Not a native shared-component library on day one
