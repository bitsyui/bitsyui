# @bitsyui/storybook

Storybook workbench for isolated component states, QA, and regression checks.

This workspace is for component state coverage and review, not the primary Bitsy UI docs surface.

Included placeholder stories:

- `QA/Components/Button`
- `QA/Components/Input`
- `QA/Components/Card`
- `QA/Components/Badge`

Theme coverage:

- Toolbar switcher for light and dark theme review

Run locally with:

```bash
pnpm --filter @bitsyui/storybook storybook
```
